# CavezzAddons
A small Chat Triggers mod for hypixel skyblock. 

Current Features:
Stonkless Stonk
Dungeon ESP
Mini bosses
Starred mobs
Blood Bonzo/Scarf/Livid
Vanquisher ESP
Auto close dungeon secret chests

All features are toggleable through the gui which you can open with /cavezz
